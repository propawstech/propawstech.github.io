Common questions

Walks aren't tracking distance

GoldenGrin needs "While Using the App" or "Always" location permission to record GPS routes. Check Settings → Privacy & Security → Location Services → GoldenGrin. If permission is granted but distance is still zero, make sure Precise Location is also enabled — without it, only an approximate region is reported and GoldenGrin can't draw a route.

Apple Health tiles show 0

The Home screen "Your steps today" and "Your active calories" tiles read from Apple Health. If you declined HealthKit permission, those tiles will show 0. Re-enable in Settings → Privacy & Security → Health → GoldenGrin. The walk tracker still works without HealthKit — it uses GPS.

How do I delete my account?

Open Settings → Privacy → Delete my account in the app. This permanently removes your account from our servers and wipes all local data. There is no soft-delete or grace period.

How do I export my data?

Settings → Privacy → Export my data produces a single JSON file containing every dog, walk, and wellness entry you've logged. You can save it to Files, AirDrop it, or email it to yourself.

Vocalisation detection isn't working

The microphone permission must be granted (Settings → Privacy & Security → Microphone → GoldenGrin). All audio is processed on-device using Apple's SoundAnalysis framework — nothing is recorded or uploaded. If your dog isn't vocalising during the walk, no events will be logged.

Why doesn't a feature I want exist?

GoldenGrin is built by a single developer focused on doing fewer things well. Email us — feature requests genuinely shape the roadmap.

About
